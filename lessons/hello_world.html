Hi and welcome to the first lesson.

In your left-side pane, there is a panel named Block Workspace. You can add and arange as many block as you need. To add blocks, click on the toolbox category, and choose your block. For now, please take a look at these 2 categories:

  • Values There are several type of values. The green one is for text, the blue one is for numbers, and the purple one is for list.
  • Input/Output There are only print and prompt here. If you want Guo to say something, you should use print, while if you want Guo to ask for a value, you can use prompt

You can combine print-block and any value block to make Guo the-panda say something for you. The current block (if you don't modify it yet) will make Guo say .... Since there is a text-value block contains ... next to print block

You can use Run button to show the result of your block immediately. You can also see python and javascript code of your block at the bottom of this pane.

Your first task is to make Guo say Hello world. Modify the current block, and click Submit block
