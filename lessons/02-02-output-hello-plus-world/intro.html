Hello + World

Untuk dilakukan:
  • Coba ubah pilihan bahasa pemrograman, dan tentukan
    • Bahasa pemrograman apa saja yang menggunakan tanda + untuk menggabungkan dua string?
    • Bahasa pemrograman apa saja yang menggunakan tanda . untuk menggabungkan dua string?
    • Bahasa pemrograman apa saja yang menggunakan tanda .. untuk menggabungkan dua string?
    • Bahasa pemrograman apa saja yang menggunakan fungsi join untuk menggabungkan dua string?
  • Gunakan blok Create text with untuk menampilkan "Hello " + "World"

Dua buah string dapat digabungkan dengan menggunakan operasi concatenation. Dua string yang digabungkan, niscaya akan membentuk sebuah string baru. Jika kelak kamu membutuhkan operasi concatenation, kamu dapat menekan kategori Operations dan memilih balok Create text with. Saat ini, balok tersebut sudah ada di lembar kerja.
