Kali ini kita meletakkan dua percabangan secara berurutan. Kebetulan di sini, nilai kondisi pertama akan selalu berlawanan dengan kondisi kedua (jika nilai > 50, maka tidak mungkin nilai <= 50, demikian sebaliknya). Amati kode Javascript yang dihasilkan.

Setelah ini, kita akan lihat bagaimana kedua kondisi tersebut bisa diringkas.
