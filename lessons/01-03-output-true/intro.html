Boolean

Untuk dilakukan: Baca seluruh petunjuk di bawah ini, dan tekan tombol evaluate.

Tipe data paling dasar yang dimiliki komputer adalah tipe data boolean. Berbeda dengan string, integer, float, dan double, boolean hanya memiliki dua kemungkinan nilai, yakni true atau false.

Tipe data boolean biasanya dipakai untuk mewakili sebuah nilai kebenaran. Misalnya, apakah hari ini hujan? Jawaban atas pertanyaan tersebut hanya bisa bernilai true atau false.
